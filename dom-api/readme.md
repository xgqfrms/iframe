## 注意事项

注意: 使用 adoptNode() 方法从一个外部文档中获取一个节点, 节点下的所有子节点都会获取到, 拷贝后在外部文档中的节点及其子节点会被删除。

注意: 使用 document.importNode() 方法从外部文档中来拷贝节点，拷贝后在外部文档中的节点不会被删除。

注意: 使用 node.cloneNode() 方法从当前文档中来拷贝节点，拷贝后在当前文档中的节点不会被删除。
